Поля запроса можно получить так:
```Js
    fetch('/api/map', {
        method: 'post',
        credentials: 'include',
        body: JSON.stringify({
            url: '/api/auth'
        })
    })
```

вернет массив
```php
    [
        'map' => ['name', 'name' ..], //Принимаемые параметры
        'url' => '/api/.../',  //урл зпроса
    ]
```
